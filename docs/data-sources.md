# Data sources

Appearance ranges are summarized by major story phase rather than treated as
an episode-by-episode cast list.

- TV anime season 1: [Official Durarara!! site](https://www.durarara.com/1st/)
- Anime x2 and OVA: [Official Durarara!! x2 site](https://www.durarara.com/)
- Light novels: [Dengeki Bunko Durarara!! series page](https://dengekibunko.jp/product/drrr/)

Character affiliations combine current, former, and plot-significant
identities so that one character can appear under multiple filters.
